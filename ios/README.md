# Recon — iOS app

A **native SwiftUI** app for the Recon internship hunt. It talks to the Recon REST
API directly and renders everything natively:

- **Today** — a small daily summary: match/pipeline/screened-out counts + top internships.
- **Internships** — the scored Summer-2027 feed, filterable by fit tier, with native cards
  showing company, tier, fit, **pay/stipend**, location, and a one-line **summary**. Tap → a
  detail view (why it fits, concerns, curriculum hook, Open posting, Track).
- **Pipeline** — applications grouped by stage with inline stage moves.
- **Plan** — the original master-plan dashboard (the one web view we keep, since it's a nice
  reference artifact).

Networking lives in `ReconAPI.swift`/`Store.swift`; it follows the endpoint picker and supports
a Cloudflare Access **service token** (Settings) for calling the API through the tunnel once an
Access policy is enforced. The Local (Tailscale) endpoint bypasses Access and needs no token.

## Open / build

The `.xcodeproj` is generated by [XcodeGen](https://github.com/yonyz/XcodeGen)
from `project.yml` (do not hand-edit the project; edit the yml and regenerate):

```bash
cd ios
xcodegen generate          # writes Recon.xcodeproj
open Recon.xcodeproj        # then set your Signing team and Run
```

CI / headless build check:

```bash
xcodebuild -project Recon.xcodeproj -scheme Recon \
  -sdk iphonesimulator -destination 'generic/platform=iOS Simulator' \
  build CODE_SIGNING_ALLOWED=NO
```

## Server endpoint

Tap the gear (top-right) to choose where the app connects:

| Mode   | URL                              | When |
|--------|----------------------------------|------|
| Tunnel | `https://recon.zacharyjcollins.com` | anywhere (Cloudflare Tunnel) |
| Local  | `http://100.91.198.28:8010`      | home / Tailscale connected |
| Custom | anything you type                | testing |

**Default is `Tunnel`** (`https://recon.zacharyjcollins.com`, set in `Recon/Config.swift`) —
works anywhere via the Cloudflare Tunnel (served by the `recon-cloudflared` sidecar in the
NUC's compose). Switch to `Local` (`http://100.91.198.28:8010`) when you want to bypass the
tunnel on the home/Tailscale network. The plain-HTTP local host is allowed via an
`NSAppTransportSecurity` exception in `project.yml` (regenerated into `Info.plist`).

## To run on a physical iPhone

Open the project, select your device, set a Development Team under
Signing & Capabilities (the bundle id is `com.zacharyjcollins.recon`), and Run.
Cloudflare Access will prompt for your identity on first load over the tunnel.
